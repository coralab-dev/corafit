# Agent Context

These documents are the small, versioned orientation layer for Codex and other coding agents. They are stable technical context, not a task log, session handoff, roadmap, or product source of truth.

GitHub keeps the repository source, pull requests, code review history, CI signals, and technical changes that belong with code.

Product context, decisions, contracts, long-form architecture notes, roadmap detail, and scope live in Notion.

Priority, ownership, and execution state live in Linear.

The concrete implementation plan for a task is provided in the chat that starts that task.

Do not store ticket or session state in these documents.
